ABOUT ME

I've been passionate about computers since I was a kid. I've always been interested in the way they work and how they can be used to solve problems.

I started creating video games at a very young age, starting with software like Scratch before moving on to Unity. I have created several game prototypes and participated in several events called game jams in which we have an established time limit to create a video game on an imposed theme.

In order to be able to realize my games I had to train, always as a self-taught person, to several art forms: 3D modeling, video editing, 2D visual art in different styles such as pixel art, game design, programming in C#.
